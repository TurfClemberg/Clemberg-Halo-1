# 🌌 Clemberg Halo – 1
**A Student-Built Smart Desktop Environment in Python**
## 🔷 What is Clemberg Halo?
**Clemberg Halo – 1** is a beginner-friendly desktop simulation system built using **Python**.
It acts like a **mini operating environment** where users can log in and access a controlled desktop system.
This project is designed to show how real computer systems handle:
* User login
* Password security
* System access control
* Modular programming
It is part of the **Clemberg Innovation Series**, focused on learning real-world computing concepts at a student level
## 🎯 Project Goal
The aim of Clemberg Halo is to:
✔ Teach how login systems work
✔ Show how operating systems control access
✔ Demonstrate modular Python programming
✔ Build the base for a future **AI-powered student OS system**
This is not just code — it is a **learning operating system prototype**
## 🧠 Key Features
🔐 **Login System**
* Username & password authentication
* Secure password input using `getpass`
🖥 **Desktop Access Simulation**
* Only correct users can access the system
* Prevents unauthorized entry
📦 **Modular Structure**
* Code divided into separate files (like real software)
* Uses Python packages and imports
🧩 **Expandable Design**
* Future modules like apps, AI assistant, tools, and system controls can be added

### File Roles

| File          | Purpose                   |
| ------------- | ------------------------- |
| `main.py`     | Starts the system         |
| `login.py`    | Handles user login        |
| `desktop.py`  | Controls desktop access   |
| `__init__.py` | Connects modules together |

---
## ⚙ How It Works
1. System starts
2. User enters username
3. Password is hidden during typing
4. System checks credentials
5. If correct → Access Granted
6. If wrong → Access Denied
Just like real computers!
## 💻 Requirements

No extra libraries needed.
